Demo & Pruebas

Diferentes formas de explorar la plataforma

Desarrollo Local

Ejecuta la web app completa en tu máquina

# Frontend
cd web_app/frontend
npm install && npm start

# Backend (terminal separado)
cd web_app/backend
pip install -r requirements.txt
uvicorn main:app --reload
Requiere Node.js 18+ y Python 3.11+
Deploy en la Nube

Despliega en GCP con un comando

# Configurar triggers
./scripts/setup-cloud-build-triggers.sh

# Deploy automático
git tag webapp-v1.0.0
git push origin webapp-v1.0.0
Requiere GCP project configurado
